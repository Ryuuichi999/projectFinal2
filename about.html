จังหวัดชลบุรี - Land Subsidence Monitoring System

เกี่ยวกับเว็บไซต์นี้

เว็บไซต์นี้จัดทำขึ้นเพื่อเป็นส่วนหนึ่งของการวิจัยเกี่ยวกับ WebMap ในการวิเคราะห์และติดตามการทรุดตัวชั้นดินในจังหวัดชลบุรี โดยนำวิธีการและเทคนิคจากซอฟต์แวร์ LiCSBAS2 ซึ่งเป็นซอฟต์แวร์วิเคราะห์ข้อมูลแบบ time series ของ InSAR ข้อมูลที่ใช้ได้มาจาก COMET-LiCS Portal

วิธีการดำเนินงาน

เริ่มต้นด้วยการดาวน์โหลดข้อมูล InSAR จาก COMET-LiCS Portal โดยใช้ Frame ID: 172A_07686_131012 ซึ่งประกอบด้วย interferograms และ coherence maps ที่ประมวลผลโดย LiCSAR processor หลังจากนั้นใช้สคริปต์ Python จาก LiCSAR Web Tools เพื่อดึงข้อมูลลงมาในเครื่องอย่างเป็นระบบ

clone LiCSBAS2 จาก GitHub โดยใช้คำสั่ง: git clone https://github.com/yumorishita/LiCSBAS2
ต่อมาใช้ไฟล์สคริปต์ batch_LiCSBAS.sh เพื่อรันกระบวนการวิเคราะห์ทั้งหมด ซึ่งประกอบด้วยขั้นตอนต่าง ๆ ตั้งแต่ 01 ถึง 16 ดังนี้:

  • ขั้นตอน 01: LiCSBAS01_get_geotiff*.py - ดึงข้อมูล GeoTIFF จากแหล่งข้อมูล (ในที่นี้ใช้ COMET_S1 เป็นแหล่งข้อมูลเริ่มต้น).
  • ขั้นตอน 02: LiCSBAS02_ml_prep.py - เตรียมข้อมูลโดยการ multilook เพื่อลดความซับซ้อน .
  • ขั้นตอน 04 : LiCSBAS04op_mask_unw.py - กรองข้อมูล unwrapped interferograms.
  • ขั้นตอน 05 : LiCSBAS05op_clip_unw.py - ตัดข้อมูลตามช่วงที่กำหนด .
  • ขั้นตอน 11: LiCSBAS11_check_unw.py - ตรวจสอบคุณภาพของข้อมูล unwrapped.
  • ขั้นตอน 12: LiCSBAS12_loop_closure.py - loop closure เพื่อตรวจสอบความถูกต้อง.
  • ขั้นตอน 13: LiCSBAS13_sb_inv.py - คำนวณการเคลื่อนที่แบบ time series โดยใช้ Small Baseline Inversion.
  • ขั้นตอน 14: LiCSBAS14_vel_std.py - คำนวณความเร็วและความแปรปรวนของการเคลื่อนที่.
  • ขั้นตอน 15: LiCSBAS15_mask_ts.py - กรองข้อมูล time series ที่มีข้อผิดพลาด.
  • ขั้นตอน 16: LiCSBAS16_filt_ts.py - กรองข้อมูลเพื่อลด noise.
หลังจากรัน batch_LiCSBAS.sh ระบบจะสร้างไฟล์ Cum_filt.h5 ซึ่งเก็บข้อมูลการทรุดตัวที่ผ่านการกรองสัญญาณรบกวนแล้ว จากนั้นใช้ Python เพื่ออ่านและแปลงเป็น GeoTIFF แยกตามช่วงเวลาเพื่อนำไปแสดงผลบน Web Map
